# 📘 Prácticas de Algoritmos

Este repositorio contiene ejercicios de práctica creados con el objetivo de **prepararme para el examen final de Algoritmos**.  
Aquí voy a ir subiendo distintos programas, ejemplos y resoluciones que me ayuden a reforzar conceptos clave.

---

## 🎯 Objetivo

    - Consolidar conocimientos de estructuras de control, funciones y recursividad.  
    - Practicar resolución de problemas algorítmicos de forma sistemática.  
    - Generar un espacio organizado para repasar antes del examen.

---

## 📂 Contenido

    - Ejercicios básicos de lógica y estructuras de control.  
    - Problemas de práctica con listas, matrices y cadenas.  
    - Implementaciones de algoritmos clásicos.  
    - Notas y ejemplos adicionales para repaso.

---

## 🛠️ Tecnologías utilizadas

    - Lenguaje: **C** (principalmente)  
    - Entorno: **Visual Studio Code**  
    - Control de versiones: **Git + GitHub**
